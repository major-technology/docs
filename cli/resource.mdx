---
title: "major resource"
description: "Manage resource attachments from your terminal."
---

Use the `resource` group to manage the databases and APIs your app can access.

| Command | Purpose |
| --- | --- |
| `major resource create` | Create a new resource via the web UI. |
| `major resource manage` | Attach or detach resources for the current app. |

## `major resource create`

```bash
major resource create
```

**When to use:** creating a new database or API resource.

**What happens:**

Opens the resource creation page in your browser.

## `major resource manage`

```bash
major resource manage
```

**When to use:** changing which resources your app is connected to.

**What happens:**

1. Fetches the current resources for your app.
2. Lets you attach or detach resources using an interactive menu.
3. Updates your local configuration to match the new settings.
